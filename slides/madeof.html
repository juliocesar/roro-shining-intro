What are slides made of?

Slides are just HTML files sitting ROOT/slides. The slide index is listed in the preso "config file" ROOT/config.json. The index defines the order in which your slides will appear.

Slides can also have styles loaded for each one individually, so if you want a set of styles applied a slide named win, add a stylesheet named ROOT/slides/win.css and it will get loaded along.

You know what else gets loaded along with slides? Slide scripts!
